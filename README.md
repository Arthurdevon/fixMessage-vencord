# 🔧 fixMessage for Vencord

[🇧🇷 Leia em Português](README.pt-BR.md)

A Vencord plugin that corrects grammar and spelling before sending messages. Built specifically for **Brazilian Portuguese Discord slang**, with optional AI provider support.

## Why not just use LanguageTool?

Because LanguageTool (and most generic tools) completely fail with BR Discord slang and contextual verb accents. This plugin has a ~400-line heuristic engine built from scratch to understand how Brazilians actually type.

Supports LanguageTool (free), OpenAI, Anthropic, or custom API endpoints.
